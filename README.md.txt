# Real Estate Buyer Segmentation using Machine Learning

## 📌 Project Overview
This project uses K-Means clustering to segment real estate buyers based on behavior, investment patterns, and demographics.

## 🧠 Techniques Used
- Data Cleaning
- Feature Engineering
- K-Means Clustering
- Elbow Method
- Data Visualization
- Streamlit Dashboard

## 📊 Features
- Buyer segmentation
- Investment analysis
- Customer behavior insights
- Interactive dashboard

## 🚀 How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
